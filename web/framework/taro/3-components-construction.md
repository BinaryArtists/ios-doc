重要切入点：

2. @tarojs/taro-alipay	taro 支付宝小程序运行时框架

5. @tarojs/taroize	taro 小程序编译器

@tarojs/plugin-babel	taro babel 编译插件

@tarojs/plugin-csso	taro css 压缩插件
